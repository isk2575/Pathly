# Pathly 🛡️

A campus safety platform for University of Houston students.

## What it does
- safer route suggestions based on lighting and reports
- Walk With Me mode with passive trip monitoring
- One-tap SOS — calls 911, texts UHPD, alerts trusted contacts
- Anonymous incident reporting
- Real-time safety heatmap

## Tech Stack
React, Node.js, PostgreSQL, Firebase, Google Maps, Socket.IO, Azure
